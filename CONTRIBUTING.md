# Contributing to Ampersand

You might want to start by reading the [wiki](https://codeberg.org/Ampersand/app/wiki/Development-Environment-Setup) to know how to get a development environment running.

Other sources of interest include [Tauri](https://tauri.app), [Vue](https://vuejs.org), [TypeScript](https://typescriptlang.org), [Rsbuild](https://rsbuild.rs/).

## AI contributions

Contributions to Ampersand **must not include** content generated by large language models (LLMs) such as, but not limited to, ChatGPT, Claude, Gemini, DeepSeek, Copilot.

This applies to the Ampersand codebase, its issue section, any and all discussions sections, pull request descriptions, the wiki, and so on.

A very slight exemption to this rule is for machine-translated sentences (not code) from your native language to English,
should you want to communicate with us and should you have problems with writing in English or Italian yourself.
