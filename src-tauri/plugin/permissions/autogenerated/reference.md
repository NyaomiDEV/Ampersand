## Default Permission

Default permissions for the Ampersand plugin

#### This default permission set includes the following:

- `allow-open-file`
- `allow-get-webkit-version`
- `allow-broadcast-event`
- `allow-list-assets`
- `allow-dismiss-splash`
- `allow-register-listener`
